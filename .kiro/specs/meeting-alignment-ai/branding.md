# 브랜딩 가이드 — Meeting Alignment AI

## 1. 프로덕트 포지셔닝

### 핵심 컨셉
"팀 인텔리전스 레이어" — 회의 보조 도구가 아니라, 각자 다른 렌즈로 같은 회의를 볼 때 AI가 그 렌즈를 맞춰주는 시스템.

### 브랜딩 키워드
- **Aligned, not uniform** — 모두가 같은 걸 보는 게 아니라, 각자의 관점에서 같은 수준으로 이해하는 것
- **Calm intelligence** — 노이즈를 줄여주는 도구가 노이즈가 되면 안 된다
- **Contextual flow** — 정보 과부하가 아닌, 맥락 있는 흐름

### 브랜드 네이밍 후보
| 이름 | 의미 | 톤 |
|------|------|-----|
| Attune | 조율하다 | 따뜻하고 정제된 |
| Synq | sync + 질문(Q) | 테크 친화적, 간결 |
| Lenz | 각자의 렌즈 | 관점 중심, 직관적 |

### 브랜드 카피 방향
- ❌ "당신의 회의를 더 똑똑하게"
- ✅ "모두가 같은 페이지에 있는 회의"
- 도구가 아니라 **상태를 약속**하는 톤

## 2. 색상 팔레트

### 디자인 원칙
- 다크모드 우선 — 회의 중 화면 공유 시 눈부심 방지, "보조 도구"로서 배경에 녹아드는 성격
- 역할별 구분이 핵심이되, 채도를 낮게 — "구분은 되지만 튀지 않는" 팔레트
- 정보 위계 명확 — 스크립트(1차) > 보조설명(2차) > 추천(3차)

## 3. 타이포그래피

### 서체 선택
- **헤딩**: Inter (또는 Pretendard) — 산세리프, 깔끔한 기하학적 형태
- **본문**: Inter (또는 Pretendard) — 가독성 최우선, 회의 중 빠른 스캔
- **코드/기술 용어**: JetBrains Mono — 모노스페이스, 기술 컨텍스트 구분

### 사이즈 스케일
| 레벨 | 사이즈 | 용도 |
|------|--------|------|
| Display | 24px / 1.5rem | 미팅 제목 |
| Heading | 18px / 1.125rem | 섹션 제목, 역할 이름 |
| Body | 14px / 0.875rem | 스크립트, 채팅, 본문 |
| Caption | 12px / 0.75rem | 타임스탬프, 메타 정보 |
| Micro | 10px / 0.625rem | 배지, 태그 |

## 4. UX 브랜딩 원칙

### Progressive Disclosure
- 보조설명 접기/펼치기를 전체 UI 철학으로 확장
- 필요한 정보가 필요한 순간에만 나타나는 느낌
- 기본 상태는 항상 "깔끔"

### Nudge, Not Push
- 추천 질문, 궤도 이탈 알림은 강제가 아닌 부드러운 제안
- 토스트보다 인라인 힌트에 가까운 형태
- 무시해도 흐름이 끊기지 않아야 함

### 정보 위계
```
1차: 실시간 스크립트 — 가장 높은 시각적 비중
2차: 보조설명 — 스크립트에 종속, 접기/펼치기
3차: 추천 질문/주제 — 사이드 패널, 부드러운 갱신
4차: 채팅 Q&A — 필요 시 접근, 독립 영역
```

### 모바일 전략
- 탭 전환 방식으로 각 영역 접근
- 탭 순서: 스크립트 → 채팅 → 추천 → 설정
- 스와이프 제스처로 탭 간 이동

## 5. 컴포넌트 톤 가이드

| 컴포넌트 | 시각적 톤 | 인터랙션 |
|----------|----------|----------|
| 브리핑 카드 | 역할 컬러 배경, 카드 형태 | 탭으로 역할 전환 |
| 스크립트 라인 | 깨끗한 타임라인, 최소 장식 | 자동 스크롤, 수동 스크롤 전환 |
| 보조설명 | 역할 컬러 좌측 보더, 접힌 상태 기본 | 클릭으로 펼치기 |
| 추천 질문 칩 | Iris 아웃라인, 필 없음 | 클릭 시 채팅 입력에 자동 채움 |
| 궤도 이탈 알림 | Amber 좌측 보더, 인라인 배너 | 자동 소멸 또는 닫기 |
| 에러 메시지 | Coral 배경 틴트, 재시도 버튼 | 재시도 클릭 |
| 녹음 인디케이터 | Sage 펄스 애니메이션 | 클릭으로 시작/중지 |
